# Карточка товара

Адаптивная карточка с заголовком, ценой, описанием и кнопкой «Купить».  
Полностью соответствует макету, работает на всех устройствах.

## 🛠 Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, медиа-запросы, hover-эффекты, CSS-переменные)

1. Склонируй репозиторий: `git clone https://github.com/твой-ник/product-card.git`
2. Открой `index.html` в браузере.
